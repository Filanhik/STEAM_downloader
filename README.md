# STEAM_downloader

Небольшой Python-скрипт для мониторинга загрузок в Steam.

Скрипт:

- проверяет активность загрузки раз в минуту в течение 5 минут  
- определяет статус загрузки (активна / на паузе)  
- извлекает AppID из логов Steam  
- сопоставляет AppID с локальными manifest-файлами для получения названия игры  
- рассчитывает объём переданных данных и скорость загрузки  

---

## Требования

- Python 3.8+  
- Библиотека `psutil`  

---

## Установка

Установите необходимую библиотеку:

```bash
pip install psutil
```
